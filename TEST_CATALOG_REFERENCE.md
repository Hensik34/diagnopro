## QUICK REFERENCE FORMULAS

### Automatically Calculated Fields
- **LDL** = Total Cholesterol - HDL - (Triglycerides / 5)
- **VLDL** = Triglycerides / 5
- **TC/HDL Ratio** = Total Cholesterol / HDL
- **Globulin** = Total Protein - Albumin
- **A/G Ratio** = Albumin / Globulin
- **eGFR** = MDRD Equation (from Creatinine)
- **eAG** = (HbA1c × 35.6) - 77.3

## SAMPLE REFERENCE RANGES

### CBC Hemoglobin
- **Male**: 13.5 - 17.5 g/dL (Critical: < 7.0 or > 20.0)
- **Female**: 12.0 - 15.5 g/dL (Critical: < 7.0 or > 20.0)

### Glucose Tests
- **FBS**: 70 - 100 mg/dL (Normal fasting)
- **PPBS**: 70 - 140 mg/dL (Normal 2h post-meal)
- **HbA1c**: 0 - 5.6% (Normal glucose control)

### Thyroid
- **TSH**: 0.4 - 4.0 mIU/L (Critical: < 0.01 or > 100)
- **Free T4**: 0.8 - 1.8 ng/dL (Critical: < 0.4 or > 3.0)

### Cardiac
- **PSA (Male 50+)**: 0 - 4.0 ng/mL (Critical: > 10.0)
- **Troponin I**: 0 - 0.04 ng/mL (Critical: > 0.1)
